Wold's Vaults is a highly customized version of the popular Vault Hunters 3 for Minecraft 1.18.2. The project consists of this (the main pack itself) and the [official mod](https://github.com/iwolfking/Wolds-Vaults-Official-Mod) which handles a lot of the new items, blocks, objectives, bug fixes, and other features of the pack.

Wold's has been in development for over 2 years now! 

A big shout-out to contributors who have come along and helped over the years, you can see them on the sidebar on this repository!

Do you want to contribute? Pull requests are always welcome! If you want to talk with me and other developers about things, joining the [Discord](https://discord.gg/woldsvaults) is highly recommended! There is no need for an application or anything fancy, all are welcome.
